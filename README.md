# www.minetest.cn

非官方Minetest中文网站，地址：[www.minetest.cn](http://www.minetest.cn).

# 特性

* 使用 Bootstrap 3.3.5 和 jQuery 2.1.4.
* 使用现代网页设计技术： `rem` 单位, `hsl` 色彩空间.
* 响应式网站 + 图标

# 工作流程

本站使用 [Jekyll](https://www.jekyll.com.cn/) 进行静态网页生成.
使用命令 `jekyll serve` 可以自动构建Jekyll网站并在本地运行
